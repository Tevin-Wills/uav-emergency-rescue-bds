## Assigned Objective

UAV Target Detection, Tracking, Following, and Fixed-Point Landing — use the onboard camera to detect the rescue target, track and follow it, and guide the UAV to a precision fixed-point landing.

- ROS 2 node design for `target_detection_tracking`.
- Fixed-point landing guidance logic.
- Simulation setup and test results.
