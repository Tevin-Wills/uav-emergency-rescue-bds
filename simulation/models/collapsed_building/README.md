# Model: Collapsed Building

This folder will contain the Gazebo Harmonic model for a collapsed building — representing the disaster environment through which the UAV must navigate.

## Planned Contents

- `model.sdf` — Gazebo SDF model definition
- `model.config` — Gazebo model metadata
- Mesh files for the rubble geometry

## Purpose

Provides realistic obstacle geometry for the path planning and obstacle avoidance module to navigate around.

## Status

Placeholder folder. Model files to be added during the simulation development phase.
