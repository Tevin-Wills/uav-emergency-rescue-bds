# Model: Survivor Marker

This folder will contain the Gazebo Harmonic model for the survivor marker — the visual target that the `target_detection_tracking` module must detect and track.

## Planned Contents

- `model.sdf` — Gazebo SDF model definition
- `model.config` — Gazebo model metadata
- Mesh or texture files if required

## Purpose

The survivor marker represents a person or rescue signal at the disaster site. It should be visually distinct enough for the camera-based detection algorithm to identify reliably.
